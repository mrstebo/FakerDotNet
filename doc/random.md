# Faker.Random

```cs
Faker.Random.Element(new[] {1, 2, 3, 4, 5})        //=> 5

Faker.Random.Element("test")                       //=> "e"

Faker.Random.Assortment(new[] {1, 2, 3, 4, 5}, 5)  // => [3, 2, 4, 5, 1]

Faker.Random.Assortment(new[] {"a", "b", "c"}, 10) // => ["b", "a", "a", "c", "b", "a", "b", "c", "c", "a"]
```
